# Sample Feature Request

```text
Use $pm-dev-workflow to build a Feed refresh feedback feature for a mobile app.
```

## What the skill should do

1. Ask what page the feature belongs to.
2. Clarify when refresh is triggered and what counts as success.
3. Define the smallest shippable version.
4. Call out loading, empty, and error states.
5. Suggest analytics events and acceptance criteria.
6. Produce the implementation plan before editing files.

## Why this helps

The user gets a product-minded response instead of immediate code.
